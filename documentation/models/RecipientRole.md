# RecipientRole

| :------------- | :--- | :------- | :----------------- |
| SIGNER         | str  | ✅       | "SIGNER"           |
| RECEIVESCOPY   | str  | ✅       | "RECEIVES_COPY"    |
| INPERSONSIGNER | str  | ✅       | "IN_PERSON_SIGNER" |
